# Support

TokenPak's open-source support is public, asynchronous, and best effort.
There is no guaranteed response or resolution time for this repository.

## Where To Ask

- Bugs, installation failures, confusing docs, and feature requests: open a
  GitHub issue in this repository.
- Security vulnerabilities: do not open a public issue. Follow
  [SECURITY.md](./SECURITY.md) instead.

## What To Include

For bugs or install failures, include:

- the TokenPak version;
- your operating system and Python version;
- the install command or command that failed;
- the full error text;
- a minimal reproduction when possible.

For feature requests, describe the workflow you are trying to complete and the
current workaround, if one exists.

## Response Expectations

Maintainers prioritize reports by impact, reproducibility, and available
maintainer capacity. Current supported versions receive priority over older
versions. Feature requests are logged for consideration, but opening a request
does not create a commitment to build it or a timeline for delivery.

TokenPak is beta software. Local services, self-hosted components, and
third-party provider behavior are not covered by an uptime or availability
guarantee.
