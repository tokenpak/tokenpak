## New Issues Found

### 🔴 Critical: Test Suite Completely Broken

**Problem:** No tests run when invoking pytest:
```
$ pytest tests/unit/test_vault_index.py -q
bringing up nodes...

no tests ran in 1.06s
```

**Root Causes:**
1. **pytest.ini has xdist args hardcoded**: `addopts = -n auto --dist=worksteal`
2. **xdist brings up nodes but can't distribute tests**
3. **Module imports fail**: `ModuleNotFoundError: No module named 'tokenpak.compression.core'`
4. **PYTHONPATH not correctly resolving**: `packages/core` not being found

**Impact:** Cannot validate code quality before deployment. CI is blind.

**Suggested Fixes:**
1. Remove `-n auto --dist=worksteal` from pytest.ini `addopts` (make xdist opt-in)
2. Fix `pythonpath` in pytest.ini or add proper package structure
3. Add `conftest.py` with explicit path setup
